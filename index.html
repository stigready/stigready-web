What You Get

Every StigReady AMI ships ready for compliance scanning from day one.

STIG Partition Layout

Separate mounts for /home, /tmp, /var, /var/log, /var/log/audit, /var/tmp with hardened mount options — the same layout across both families.

SSH & FIPS Hardened

Password auth disabled, root login prohibited, host keys regenerate on first boot, no pre-installed keys. STIG images run FIPS 140-3 validated crypto, verified at build time.

Audit-Grade Evidence

Every build produces an OpenSCAP report + ARF, a POA&M, an SBOM (SPDX + CycloneDX), and a CVE scan — a complete, reproducible compliance record per image.

AWS-Native

IMDSv2 enforced. SSM Agent pre-installed. cloud-init handles EC2 key-pair injection. Boot-tested on real Nitro EC2 before any image ships. EBS-backed, HVM, x86_64.
